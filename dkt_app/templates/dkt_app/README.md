1. dkt_app/templates/dkt_app/student_list.html

  学生学习情况查看页面
  - 继承自 page_base.html
  - 功能：
    - 展示学生列表（分为"所教班级学生"和"其他学生"两类）
    - 支持实时搜索过滤（按姓名或ID）
    - 显示学生头像、姓名、ID、微信昵称
    - 点击卡片可跳转到查看该学生的知识掌握度详情
    - 包含空状态和无搜索结果的提示
  - 样式特点：使用CSS变量定义主题色，卡片式布局，支持响应式设计，带动画效果

  2. dkt_app/templates/dkt_app/my_mastery.html

  DKT知识掌握度分析页面
  - 继承自 backend_sys_base_page.html，使用自定义模板标签 dkt_extras
  - 功能：
    - 学生信息卡片（头像、姓名、ID、知识点数量）
    - 学习摘要仪表盘（总练习次数、正确率、当前掌握度、进步趋势）
    - 知识点掌握度趋势折线图（支持显示/隐藏各知识点、平均趋势线）
    - 双雷达图对比（初始状态 vs 当前状态）
    - 知识点详细分析列表（掌握度进度条、趋势箭头、练习次数、学习建议标签）
    - 个性化学习建议模块（自动分析最强/最弱知识点、进步最快/下滑的知识点）
  - 技术栈：使用 Chart.js 3.9.1 绑制图表，自定义CSS样式，支持响应式布局
  - 支持教师视图模式（is_teacher_viewing），显示提示和返回按钮

  这两个模板是为 DKT（深度知识追踪）模块设计的教师/管理员视图，用于查看学生的学习数据和分析报告。
